# Deployment

## Vercel (recommended)

1. Create a new project on Vercel and import this GitHub repo.
2. (Optional) Set env var:

```bash
NEXT_PUBLIC_SOLANA_RPC_URL=
```

3. Deploy.

## Custom domain

After the first successful deploy:
- Add your domain in Vercel → Project → Settings → Domains
- Follow Vercel DNS instructions
